# 《Cosmos》客户端

用于可视化[Cosmos](https://github.com/NGMAAAYO/Cosmos)的比赛回放的工具。基于Unity 2021开发。

## 使用方法

在[Release](https://github.com/NGMAAAYO/Cosmos-Client/releases/latest)中下载最新版本的客户端，将rpl文件放置在`Cosmos_Data\Replays`文件夹中。
